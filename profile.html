기획 의도

기존 포인핸드는 다양한 정보를 제공하지만 메인 화면의 정보량이 많아 원하는 기능을 빠르게 찾기 어렵고, 입양 관련 정보가 여러 영역에 분산되어 사용자 흐름이 끊기는 문제가 있었습니다.

이에 모바일 퍼스트 환경을 기준으로 정보 구조를 재설계하고, 보호소 찾기와 입양 콘텐츠의 접근성을 강화해 탐색부터 입양까지 자연스럽게 이어지는 경험을 제공하는 것을 목표로 리뉴얼을 진행하고 있습니다.

개선 방향

  • 메인 화면의 정보 우선순위를 조정해 핵심 기능을 빠르게 탐색
  • 보호소 찾기와 입양 콘텐츠를 중심으로 정보 구조 재설계
  • 탐색부터 입양까지 끊김 없는 사용자 흐름 구성
  • 모바일 환경에 최적화된 직관적인 인터페이스 설계

진행 상태

사용자 흐름과 정보 구조를 정리하고 모바일 UI를 설계하고 있습니다.

기획 의도

  • TMDB API를 활용하여 영화 목록, 상세 정보, 출연진 정보를 제공
  • React state와 useEffect를 활용해 데이터를 화면에 동적으로 표시
  • localStorage를 활용하여 관심 영화(찜하기) 저장 및 해제 기능 구현
